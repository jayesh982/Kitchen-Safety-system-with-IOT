# Kitchen Safety System with IoT

An IoT-based smart kitchen safety system designed to detect gas leakage, abnormal sounds, and monitor kitchen conditions using embedded systems and TinyML.

## Features

- Gas leakage detection
- Smart stove monitoring
- IoT-based safety alerts
- ESP32 implementation
- STM32 implementation
- Raspberry Pi Pico implementation
- TinyML sound classification

## Technologies Used

- Arduino UNO
- ESP32
- STM32
- Raspberry Pi Pico
- C/C++
- MicroPython
- TinyML
- Wokwi Simulator

## Project Structure

1_tinkercad_arduino_uno  
→ Arduino gas monitoring system

2_wokwi_esp32  
→ ESP32 IoT implementation

3_wokwi_stm32  
→ STM32 implementation

4_wokwi_pico  
→ Raspberry Pi Pico implementation

5_tinyml  
→ Machine learning based sound detection
